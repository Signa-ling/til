# Todo Macro

## 概要

Rust のマクロの一種で、今後実装する旨を示す。
宣言しておくことで型チェックを通過させるが、実行時にはパニックを起こす。この挙動により型システムを満たしつつ未実装であることを伝える。

```rust
fn hoge() -> bool {
    // 本来は bool 型 を返す処理が必要だが、まだ未実装だとする
    // そのため普通なら関数の返り値の型が合わない旨が示されるが、以下を示すことで型チェックを通過する
    todo!()
}
```
